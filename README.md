# Studybook

This Android project is still a work in progress. This app is designed to become a valuable scheduling and planning tool for students with busy schedules. It is being developed using the Test Driven Development methodology and is utlizing the latest modern technologies and architecture.

#### Main Libraries so far

* [Dagger 2](https://github.com/google/dagger)
* [RxJava 2](https://github.com/ReactiveX/RxJava)
* [RxAndroid](https://github.com/ReactiveX/RxAndroid)
* [Firebase](https://github.com/firebase/)
* [ObjectBox](https://github.com/objectbox/objectbox-java)
* [Dart](https://github.com/f2prateek/dart)
* [ButterKnife](https://github.com/JakeWharton/butterknife)
* [LeakCanary](https://github.com/square/leakcanary)
* [Mockito](https://github.com/mockito/mockito)
